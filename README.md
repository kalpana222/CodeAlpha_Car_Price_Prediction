# 🚗💰 Car Price Prediction with Machine Learning

# 📖 Project Overview

This project aims to predict the selling price of used cars using Machine Learning techniques. It involves data cleaning, exploratory data analysis (EDA), feature engineering, data preprocessing, model training, evaluation, and price prediction.

Three regression models are trained and compared to determine the most suitable model for predicting car prices based on various vehicle features.

---

# 🎯 Project Objective

The main objectives of this project are:

- Predict the selling price of used cars.
- Perform data cleaning and preprocessing.
- Explore the dataset through visualizations.
- Train multiple regression models.
- Compare model performance using evaluation metrics.
- Predict prices for new car examples.

---

# 📊 Dataset Information

The dataset contains information about used cars, including:

- Car Name
- Manufacturing Year
- Present Price
- Selling Price
- Driven Kilometers
- Fuel Type
- Selling Type
- Transmission
- Number of Previous Owners

# 🤖 Machine Learning Models

The following regression models were implemented:

- 📈 Linear Regression
- 🌳 Decision Tree Regressor
- 🌲 Random Forest Regressor

Each model was trained and evaluated to identify the best-performing regression algorithm.

---

# 📏 Evaluation Metrics

The models were compared using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

Among all three models, **Linear Regression** achieved the best overall performance on this dataset and was selected as the final model for prediction.

---

# 🚗 Price Prediction

The trained Linear Regression model was used to predict the selling prices of different cars based on their features.

The project demonstrates how machine learning can estimate vehicle resale values using historical data.
